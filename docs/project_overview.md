# Project Overview

## Title

Patient Discharge Experience & Follow-Up Support Analytics

## Summary

This project connects public CMS HCAHPS patient experience data with a Qualtrics follow-up survey design and Excel reporting workflow. The repository is organized as a portfolio-ready foundation for demonstrating survey design, data preparation, dashboard reporting, and insights communication.

## Analytical Focus

- Patient discharge experience
- Follow-up support needs
- Patient experience measure organization
- Survey-to-reporting workflow design
- Clear documentation for reproducible analysis

## Core Components

- CMS HCAHPS data source documentation
- Qualtrics follow-up survey instrument
- Survey logic and embedded data map
- Data dictionary starter
- Python data preparation workflow
- Excel dashboard workbook
- Insights brief structure

## Current Scope

- Dataset: CMS HCAHPS Hospital patient survey data
- Geography: Florida hospitals
- Reporting period: 07/01/2024-06/30/2025
- Processed file: `data/processed/hcahps_patient_experience_fl_clean.csv`
- Dashboard workbook: `excel_dashboard/patient_discharge_experience_dashboard.xlsx`
- Qualtrics role: follow-up survey design only; no Qualtrics response export is included
