# Project Overview

## Title

Patient Discharge Experience & Follow-Up Support Analytics

## Summary

This project connects public CMS HCAHPS patient experience data with a Qualtrics follow-up survey design and Excel reporting workflow. The repository is organized as a portfolio-ready foundation for demonstrating survey design, data preparation, dashboard planning, and insights communication.

## Analytical Focus

- Patient discharge experience
- Follow-up support needs
- Patient experience measure organization
- Survey-to-reporting workflow design
- Clear documentation for reproducible analysis

## Core Components

- CMS HCAHPS data source documentation
- Qualtrics follow-up survey instrument
- Survey logic and embedded data map
- Data dictionary starter
- Python data preparation placeholder
- Excel dashboard structure
- Insights brief template
